## Instructions

When asked to evaluate a source, follow these steps:

1. **Domain Assessment**
   - Is it an established publication, a government/educational domain (.gov, .edu), a known industry source, or an unknown blog?
   - Note if the domain has editorial standards or is user-generated content.

2. **Recency Check**
   - What is the publication date? Is it still relevant to the topic?
   - Flag if the information is likely stale for the subject matter.

3. **Cross-Reference**
   - Compare the key claims against other sources found during research.
   - Note consensus, contradictions, or claims unique to this source.

4. **Credibility Rating**
   - Provide a rating: **High** / **Medium** / **Low** / **Unverifiable**
   - Briefly explain the rationale.

5. **Report**
   - Return the evaluation as a concise fact that can be included in the final report.
   - If the source is Low or Unverifiable, recommend finding a corroborating source before using the information.
